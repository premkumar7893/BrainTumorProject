AI-Based Network Threat Identification

A final-year engineering project demonstrating the application of machine learning to network security. The system uses both supervised and unsupervised learning to detect attacks in real-time network traffic.

🔬 Technology Stack

DatasetCICIDS2017 (Canadian Institute)
SupervisedRandom Forest Classifier
UnsupervisedIsolation Forest
BackendPython / Flask API
Librariesscikit-learn, pandas, matplotlib
FrontendHTML5 / CSS3 / Vanilla JS

⚙️ How It Works

01
Load & Preprocess

Network traffic data from CICIDS2017 is cleaned, encoded, and scaled using StandardScaler.

02
Train Models

Random Forest (100 trees) for supervised classification. Isolation Forest for anomaly detection.

03
Real-Time Predict

Incoming packets are scaled and classified. Attack traffic triggers ALERT status immediately.
